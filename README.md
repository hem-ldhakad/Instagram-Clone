# Instagram Clone Project
Built a full-stack Instagram Clone using React, Node.js, Express, MongoDB, and ImageKit for creating and displaying image posts.

## Project Setup
Configured separate frontend and backend projects and organized the project structure.

## Backend Setup
Built an Express server and connected it to MongoDB using Mongoose.

## Database Models
Created Mongoose schemas and models for storing post data.

## Image Upload
Integrated **Multer** to handle image uploads and **ImageKit** for cloud image storage.

## REST API
Developed backend API endpoints to upload and retrieve posts.

## React Frontend
Built the user interface using React components and Hooks.

## Axios
Used **Axios** to communicate with the backend APIs for uploading and fetching posts.

## Display Posts
Fetched post data from the backend and rendered it dynamically in the feed.

## Form Handling
Implemented form submission to create new posts with images and captions.

## State Management
Managed application state using React Hooks (`useState` and `useEffect`) to update the UI dynamically.
